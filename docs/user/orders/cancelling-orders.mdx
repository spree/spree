There may be times when you need to cancel an order, whether to accommodate a customer request, flag a fraudulent transaction, or resolve issues with inventory or payments.

Spree allows you to cancel orders directly from the admin dashboard in just a few steps.

## When You Can Cancel an Order

You can cancel an order as long as **no shipment has been shipped yet**. In practice, cancellation is available when every shipment on the order is in one of these [shipment states](/user/orders/order-states#shipment-states):

- No shipment yet
- `Pending`
- `Ready`
- `Backorder`
- `Cancelled`

Once a shipment moves to `Shipped` — or to `Partial`, which means at least one shipment has already gone out — the **Cancel Order** action is no longer offered. For those orders, process a [<u>refund</u>](https://spreecommerce.org/docs/user/orders/refunding-orders) or a [<u>return</u>](https://spreecommerce.org/docs/user/returns/returns-processing) instead.

<Note>
Cancellation is available on completed orders that haven't shipped, not only on incomplete carts. If you're working with the order state machine directly and see an error such as `Cannot transition state via :cancel from :complete`, the order itself is still cancelable through `Spree::Order#canceled_by` (or the admin **Cancel Order** action). That error is raised by the checkout state machine, which doesn't define a `:cancel` event from the `complete` state — the cancel flow lives on the order, not on the checkout transitions.
</Note>

## How to Cancel an Order

Navigate to the **Orders** tab of the admin dashboard.

![Spree Commerce Orders tab showing order states](/docs/images/user/orders/cancelling-orders/1-cancelling-orders.png)

Find and click on the order you want to cancel to open its detail page.

![An order details page in the Spree Commerce admin dashboard](/docs/images/user/orders/cancelling-orders/2-order-details.png)

In the top-right corner, click the **three-dot menu**, then select **Cancel Order** from the dropdown.

![Cancelling an order in the Spree Commerce admin dashboard](/docs/images/user/orders/cancelling-orders/3-cancel-order.png)

Once confirmed, the order will be cancelled and its status will be updated automatically.

<Warning>
  Cancelling an order is irreversible.
</Warning>

## What Happens After Cancellation

### Order Status

- The order’s shipment state will be updated to Cancelled.
- The payment state may also change to Void or Refunded, depending on whether payment has already been captured.

### Payment Refunds

Refunds are not always automatic. Whether the payment is refunded depends on:

- Your payment service provider (PSP)
- The payment method used
- Whether the payment had already been captured

If the payment state does not change to Refunded, you'll need to issue a refund manually.

Learn more about refunds and returns here:

- [<u>Refunding Orders</u>](https://spreecommerce.org/docs/user/orders/refunding-orders)
- [<u>Returns Processing</u>](https://spreecommerce.org/docs/user/returns/returns-processing)
