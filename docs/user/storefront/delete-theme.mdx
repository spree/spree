If you need to remove a theme from your store - for example, to clean up unused drafts or simplify theme management - you can do so easily from the Spree dashboard.

### How to Delete a Theme

![Deleting a storefront theme in Spree Commerce](/docs/images/user/storefront/themes/delete-theme/1-theme-deletion.png)

1. Navigate to the **Storefront** tab of the admin dashboard.
2. Locate the theme you want to delete.
3. Click the three-dot menu (`â‹¯`) on the right-hand side of the theme card.
4. Select **Delete** from the dropdown menu.

<Note>Only **Draft** themes can be deleted. Published themes must be unpublished or replaced before they can be removed.</Note>
