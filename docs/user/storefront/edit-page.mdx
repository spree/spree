From time to time, you may find it necessary to update the products or information that's displayed on your custom pages, perhaps even the layout itself.

To edit a page, navigate to the Pages tab of the admin dashboard, found under Storefront.

<img height="359" width="624" src="https://lh7-rt.googleusercontent.com/docsz/AD_4nXdbBMTBOLrsWwdNfeN14M5PqC4k442AbUFY_kf0VctQS1rB1X9Z_nnVcOZXXrvAywwLGCzBwMzDC2Dh92URbvLky_KwVKTcXGF-rJb9spwpnFs3OF4In-iOvwcj22-6jzTjYSk3?key=Rhw7vMyBZx_anfGhfR9KcWKj" />

Locate the page you’d like to edit and click on it. This will open up the Theme Editor for that specific page.

<img height="359" width="624" src="https://lh7-rt.googleusercontent.com/docsz/AD_4nXfgINHDkP_A9VBB0YnoD2K1jl0RsIs5SFNw027psYoLWTZIjsf0DjclF3M39phdn67wHAlRs8isj9kNrl-saoSBDIzNzl5ho3P3bSEGtOr3JbZf4NUS1Ia08xWMuMCG2WDb_v_3?key=Rhw7vMyBZx_anfGhfR9KcWKj" />

Here you can modify the page as you would with any other page in the Theme Editor as described in this support article: \{\{LINK TO THEME EDITOR}}

<img height="359" width="624" src="https://lh7-rt.googleusercontent.com/docsz/AD_4nXfyf58YSq-aKHagv3HA8mwo9LshVemVnZrgVak81IW_d649AVwzXtAuGfCTPKhThY6wJl1kRn6ZALaZ64_0Y1nKQob0E0DIURJ6Ls5lENQTdk0V83HzeCLU0NZ2KrMXW8w0Vjov?key=Rhw7vMyBZx_anfGhfR9KcWKj" />

Important: Make sure to click ‘Publish’ once you’re done editing the page, otherwise any changes you’ve made will not be saved when you navigate away from the Theme Editor.

## Page Settings

In the top bar of the Theme Editor, you’ll notice the ‘Page Settings’ button. 

<img height="401" width="624" src="https://lh7-rt.googleusercontent.com/docsz/AD_4nXeRuGVYdSBCdhdMQ8zAzXNQptWAZxUGzqQ0fjHHmsnonYkos3aMmyigMU9RO6_JuY01fUuIrvDuiYoxOLvqIJFoDFDvoBLvtHw0huhuRRysAJXJwWU5jPSiIJ6_RFnfD_NVzJgYig?key=Rhw7vMyBZx_anfGhfR9KcWKj" />

Clicking on this will open up a window that allows you to change the page name, meta title and description, and the URL slug. There is also an option to delete the page.

<img height="399" width="624" src="https://lh7-rt.googleusercontent.com/docsz/AD_4nXduY5WRw3ovUXikpTwhYQGcYcKwK-f5DC_8Q-mWd54ifGAgp57oY63I7ecv2KgLOuPBYOnLd055IjSbGGYrPU2g1MZ1PL6Vm77MmVgP30YnCkpSK-3UA8sWnnskir7B8brUQNx9fw?key=Rhw7vMyBZx_anfGhfR9KcWKj" />

Simply edit the information as required and click the blue ‘Save’ button at the bottom of the window.
