<Info>
  Before proceeding to upgrade, please ensure you're at [Spree 4.8](/developer/upgrades/4.x-to-4.8)
</Info>

## Upgrade steps

### 1. Update gems

```bash
bundle update
```

### 2. Install and run missing migrations

```bash
bin/rake spree:install:migrations && bin/rails db:migrate
```
