<Info>
  Before proceeding to upgrade, please ensure you're at [Spree 4.9](/developer/upgrades/4.8-to-4.9)
</Info>

## Upgrade steps

### 1. Update gems

```bash
bundle update
```

### 2. Install and run missing migrations

```bash
bin/rake spree:install:migrations && bin/rails db:migrate
```
